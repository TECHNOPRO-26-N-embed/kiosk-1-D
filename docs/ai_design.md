## 2. データ構造（自然言語）

| 情報名 | 内容 |
|-------|------|
| 商品ID | 商品を識別する番号 |
| 商品名 | 商品の名称 |
| 価格 | 税抜価格 |
| 税率区分 | 店内 / 店外 / 免税 |
| カテゴリ | 商品分類 |
| カート内商品リスト | 商品ID・数量の組み合わせ |
| 合計金額 | 税込総額 |
| 税額内訳 | 税率ごとの税額 |
| 決済方法 | 現金など |
| 投入金額 | 現金決済時の金額 |
| 取引ID | 会計ごとの識別番号 |
| 購入日時 | 取引日時 |

## 3. CSV形式（更新済）

### 3.1 商品マスタCSV


商品ID,商品名,価格,税率区分,カテゴリ

---

### 3.2 取引ログCSV


取引ID,日時,決済方法,合計金額,税額内訳,購入商品一覧

---

### 3.3 カート明細CSV


取引ID,商品ID,商品名,単価,数量,小計,税率区分
